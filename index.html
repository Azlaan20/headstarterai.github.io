Closed-Loop Position Control for DC Motor with Shaft Encoder

Objectives:

  • DC Motor with Shaft Encoder: Integration of a shaft encoder to provide feedback on the motor's position.
  • Drive Control with H-Bridge: Utilization of an H-Bridge circuit for bidirectional motor control.
  • Microcontroller for PWM Generation: Arduino Uno is employed for generating Pulse Width Modulation (PWM) signals.
  • Microcontroller Interface with Simulink: Interface setup for real-time control algorithm implementation and monitoring using Simulink.

Project Overview:

  • Modeling the DC Encoder Motor: Derivation of the motor's transfer function to understand its dynamic behaviour.
  • Controller Design: Utilization of MATLAB's root locus and "sisotool" for designing proportional (P), integral (I), derivative (D), and PID controllers.
  • Arduino Implementation: Deployment of Arduino Uno to execute the designed control algorithms and generate PWM signals for motor speed control.
  • Real-time Monitoring: Feedback from the shaft encoder is processed in real-time to adjust PWM signals and achieve precise motor position.

Simulation and Validation:

  • MATLAB Simulations: Validated the designed controllers' performance in achieving desired system responses.
  • Real-world Tests: On the Arduino platform demonstrated the effectiveness of the PID controller in minimizing position errors and improving stability.

Technology Stack: MATLAB, Simulink, Arduino Uno, H-Bridge Circuit, DC Motor with Shaft Encoder, PWM Signals, Root Locus Method, PID Control

Design and Implementation of Filters for EEG Signals using MATLAB

Project Duration: May 2024 - Jun 2024

Responsibilities:

  • Processed EEG data using MATLAB.
  • Implemented filters (notch, high pass, low pass, band pass) to isolate EEG frequency bands: delta, theta, alpha, sigma, and beta.
  • Applied advanced DSP techniques to cleanse EEG signals and prepare them for analysis.
  • Conducted PSD analysis to identify and remove 50Hz or 60Hz line noise using notch filters.
  • Designed custom filters for each EEG frequency band to enhance signal clarity.
  • Validated filtering efficacy through FFT analysis and isolated brain activity bands.

Project Results:

The filters effectively removed line noise and isolated specific EEG frequency bands, significantly improving signal fidelity. The sequential filtering refined signal quality, making the EEG data more interpretable and insightful.

Conclusion:

This project underscores the critical role of DSP in optimizing EEG signal quality, paving the way for more accurate brain activity analysis and neuroscientific research.

Technology Stack: MATLAB, Digital Signal Processing, EEG Signal Processing, Notch Filters, Band Pass Filters

Design and Implementation of a Microstrip Inset Feedline Square Patch Antenna for 2.4 GHz

Project Duration: May 2024 - Jun 2024

Responsibilities:

Project Highlights:

Design Challenges:
  • Calculated precise antenna dimensions and effective permittivity (εr) to ensure optimal performance at 2.4 GHz.
Simulation and Validation:
  • Utilized HFSS software for in-depth simulations to validate design parameters, including return loss, bandwidth, and gain characteristics.
PCB Fabrication:
  • Implemented etching techniques on FR4 substrate to translate the antenna design into a functional PCB layout, ensuring accuracy in component placement and signal integrity.
Testing and Optimization:
  • Conducted rigorous testing phases to measure antenna performance metrics such as return loss, bandwidth efficiency, and directivity.
  • Made iterative optimizations to fine-tune the design for maximum efficiency.

Key Learnings:

RF Engineering Insights:
  • Gained an understanding of the intricacies of microstrip feedline topology for impedance matching and effective signal transmission.
Performance Metrics:
  • Achieved a -36dB return loss at 2.42 GHz with an impressive bandwidth of 170 MHz (-3 dB) and 60 MHz (-10 dB).
  • Recorded a far-field gain of 2.23 dB.
Technical Expertise:
  • Mastered PCB fabrication using double-sided copper-clad boards and ferric chloride etching solutions, ensuring precision in circuitry layout and connectivity.

This project underscores our commitment to excellence in RF design and engineering, combining theoretical knowledge with practical application to deliver a high-performance microstrip antenna solution.
